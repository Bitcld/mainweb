Infraestructura autosuficiente con servicios en la nube

Este proyecto reúne contenedores, seguridad, monitorización y servicios cloud para demostrar la capacidad de desplegar y gestionar una infraestructura completa en producción.

Objetivos del proyecto

  • Implantar servicios esenciales en un servidor Linux usando Docker y contenedores.
  • Garantizar seguridad mediante VPN, cifrado y gestión de accesos.
  • Centralizar almacenamiento, contraseñas y correo profesional.
  • Integrar monitorización y gestión remota.
  • Explorar el uso de Inteligencia Artificial local (Ollama) en la infraestructura.

Tecnologías utilizadas

Linux Server
Docker y Docker Compose
Cloudflare
Tailscale VPN
Nextcloud
AdGuard Home
Uptime Kuma
Netdata
Vaultwarden
Caddy
IA (Local y API)
Zoho Mail
